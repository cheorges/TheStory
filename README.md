## Quick Start

Run this command `./gradlew build`

## Language

### Datatype

* String Literal
* Number (*round half up*)

### Variable

A varable can be set of two ways:

```
Let x be 1
Put 1 into x
```

A variable ...

* ... has no fix datatype.
* ... can have the following prefix: `a`, `an`, `the`, `my` and `your`
* ... can also be a name like `Black Pearl`.
